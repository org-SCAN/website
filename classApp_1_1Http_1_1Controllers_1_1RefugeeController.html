static App\Http\Controllers\RefugeeController::apiGetPerson ( Request  $request,
Crew  $crew = null 
)
static

This function is used to return the get person API request. It should be used only by the API. It returns the list of all the person for a given crew. By default, the crew of the user, if a crew id is given, it will return the person of the given crew. It never returns the person that are not in the given crew. The person must have get permission to be returned.
